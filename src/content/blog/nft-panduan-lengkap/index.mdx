NFT (Non-Fungible Token) telah mengubah cara kita memandang kepemilikan digital. Dari seni digital yang terjual jutaan dollar hingga item gaming yang dapat diperdagangkan, NFT membuka era baru ekonomi digital yang memungkinkan true ownership atas aset digital.

## Apa itu NFT?

**Non-Fungible Token (NFT)** adalah token digital unik yang merepresentasikan kepemilikan atas aset digital atau fisik tertentu. Berbeda dengan cryptocurrency yang fungible (dapat ditukar 1:1), setiap NFT memiliki karakteristik unik yang membuatnya tidak dapat digantikan.

### Karakteristik Utama NFT:

- **Unique**: Setiap NFT memiliki identifier unik
- **Indivisible**: Tidak dapat dibagi menjadi bagian kecil
- **Verifiable**: Kepemilikan dapat diverifikasi di blockchain
- **Transferable**: Dapat dipindah tangankan antar wallet
- **Programmable**: Dapat memiliki utility dan fungsi khusus

## Standar NFT Populer

### ERC-721 (Ethereum)
- **Standard pertama**: Untuk unique tokens
- **One token per contract call**: Individual minting
- **Most popular**: CryptoPunks, Bored Ape
- **Gas intensive**: Mahal untuk batch operations

### ERC-1155 (Ethereum)
- **Multi-token standard**: Support fungible + non-fungible
- **Batch operations**: Mint multiple dalam satu transaksi
- **Gas efficient**: Lebih murah untuk gaming
- **Complex but powerful**: Flexible use cases

### SPL (Solana)
- **Low fees**: Mint costs ~$0.01
- **Fast transactions**: Sub-second confirmations
- **Growing ecosystem**: Magic Eden, Solanart
- **Compressed NFTs**: Even cheaper untuk mass minting

## Cara Kerja NFT

### 1. Smart Contract
NFT disimpan dalam smart contract yang mengatur:
```solidity
// Simplified ERC-721 structure
contract MyNFT {
    mapping(uint256 => address) private _owners;
    mapping(uint256 => string) private _tokenURIs;

    function mint(address to, uint256 tokenId, string memory uri) {
        _owners[tokenId] = to;
        _tokenURIs[tokenId] = uri;
    }
}
```

### 2. Metadata Storage
NFT contract menyimpan link ke metadata (JSON file):
```json
{
  "name": "Cool NFT #1234",
  "description": "A very cool collectible",
  "image": "https://ipfs.io/ipfs/QmYoU...",
  "attributes": [
    {
      "trait_type": "Color",
      "value": "Blue"
    }
  ]
}
```

### 3. Ownership Verification
Blockchain records membuktikan siapa yang memiliki NFT:
- **Transaction history**: Track perpindahan ownership
- **Current owner**: Address yang currently memiliki token
- **Authentic verification**: Tidak dapat dipalsukan

## Cara Membeli NFT

### 1. Setup Wallet
**MetaMask** untuk Ethereum NFTs:
- Install browser extension
- Backup seed phrase securely
- Add networks (Polygon untuk cheaper options)

### 2. Fund Wallet
- Buy ETH dari exchange Indonesia (Indodax, Pintu)
- Transfer ke MetaMask wallet
- Keep extra ETH untuk gas fees

### 3. Connect ke Marketplace
- Visit OpenSea atau platform pilihan
- Connect wallet
- Browse collections atau specific NFTs

### 4. Make Purchase
**Fixed Price**:
- Click "Buy Now"
- Confirm transaction di wallet
- Pay gas fees + NFT price

**Auction**:
- Place bid amount
- Wait for auction end
- Claim NFT jika winning bid

### 5. Verify Ownership
- Check wallet untuk NFT
- Verify di blockchain explorer
- Set sebagai profile picture jika desired

## Cara Membuat NFT

### 1. Create Digital Asset
- **Art**: Use Photoshop, Procreate, atau AI tools
- **Music**: Record atau produce dalam DAW
- **Video**: Edit dengan professional software
- **3D Models**: Blender, Maya untuk metaverse assets

### 2. Choose Blockchain
**Ethereum**: Most established, higher fees
**Polygon**: Layer 2, lower fees
**Solana**: Very cheap, growing ecosystem
**Tezos**: Eco-friendly, art-focused

### 3. Upload ke Marketplace
**Lazy Minting** (recommended):
- Upload ke OpenSea
- NFT created saat first sale
- No upfront gas costs

**Direct Minting**:
- Pay gas fees immediately
- NFT exists di blockchain instantly
- Higher initial cost

### 4. Set Details
- **Name & Description**: Clear, compelling copy
- **Properties**: Traits yang relevant
- **Royalties**: Future sales percentage (typically 5-10%)
- **Collection**: Group related NFTs

### 5. List for Sale
- **Fixed price**: Immediate purchase option
- **Auction**: Let market determine value
- **Bundle**: Multiple NFTs together
